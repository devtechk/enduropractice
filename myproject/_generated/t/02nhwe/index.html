Welcome!

Here is a simple boiler-plate to start coding sites and edit on the fly (auto css compiling)! Include:

  • gulp
  • browser-sync
  • gulp-sass
  • nodemon
  • sass

Requirements

Just install node.js easy, click

  • node.js
  • npm

Install

Be sure to have node.js (and npm) installed on your computer, then go to the directory with your terminal and run:

  • npm install

Use it!

To run go to the directory open a terminal window and run:

  • npm run start
